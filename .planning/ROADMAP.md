# Roadmap

## Phase 1: Setup e Base de Dados (MVP Inicial)
**Goal:** Estruturar o projeto web, configurar os dados com os times principais do EA FC 26 e criar a tela de cadastro de jogadores.

## Phase 2: Lógica de Campeonatos e Chaveamento
**Goal:** Implementar o motor de geração de tabelas e brackets (Fase de Grupos e Mata-mata) baseado nos inscritos.

## Phase 3: Painel do Administrador
**Goal:** Criar a interface para o admin lançar os resultados das partidas, atualizando o campeonato em tempo real.

## Phase 4: Visualização (Modo Leitura)
**Goal:** Interface mobile-friendly para que os jogadores acompanhem o torneio de seus celulares.
