# Requirements

## Epic: Gerenciador de Torneios EA FC 26

### MVP (Minimum Viable Product)
- REQ-01: O sistema deve permitir criar um novo torneio escolhendo o formato (ex: Mata-mata ou Grupos + Mata-mata).
- REQ-02: O sistema deve permitir o cadastro de participantes e a escolha de times do EA FC 26.
- REQ-03: Apenas o Administrador poderá lançar e editar resultados das partidas.
- REQ-04: O sistema deve gerar e atualizar automaticamente as chaves e tabelas de classificação após cada partida lançada.
- REQ-05: O sistema deve carregar uma lista estática (ou via API simplificada) dos elencos/times principais disponíveis no EA FC 26.

### Futuro (Out of Scope for MVP)
- REQ-F1: Autenticação de jogadores para que os próprios reportem resultados.
- REQ-F2: Matchmaking online.
- REQ-F3: Histórico de estatísticas por jogador (Gols, Assistências, etc) - foco inicial apenas nos placares gerais.
